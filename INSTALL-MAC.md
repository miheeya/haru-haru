# 하루하루 - macOS 설치 안내

## 다운로드

1. 아래 링크에서 **haru-haru-v0.2.0-mac.zip** 파일을 다운로드합니다.
   - https://github.com/miheeya/haru-haru/releases/latest

## 설치 방법

1. 다운로드한 `haru-haru-v0.2.0-mac.zip` 파일을 더블클릭하여 압축을 풉니다.
2. 압축 해제된 폴더 안에서 `하루하루.app` 파일을 **응용 프로그램(Applications)** 폴더로 드래그합니다.

## 처음 실행 시 (보안 경고 해제)

macOS는 App Store 외부에서 다운로드한 앱을 기본적으로 차단합니다.
처음 실행할 때 아래 절차를 따라주세요.

### 방법 1: 마우스 오른쪽 클릭으로 열기
1. `하루하루.app`을 **마우스 오른쪽 클릭** (또는 Control + 클릭)합니다.
2. 메뉴에서 **열기**를 선택합니다.
3. "확인되지 않은 개발자" 경고가 나타나면 **열기** 버튼을 클릭합니다.
4. 이후부터는 일반적으로 더블클릭으로 실행할 수 있습니다.

### 방법 2: 시스템 설정에서 허용하기
1. `하루하루.app`을 더블클릭합니다.
2. "손상되었기 때문에 열 수 없습니다" 또는 "확인되지 않은 개발자" 경고가 나타납니다.
3. **시스템 설정 > 개인정보 보호 및 보안**으로 이동합니다.
4. 화면 하단에 "하루하루" 앱이 차단되었다는 메시지가 보입니다.
5. **확인 없이 열기** 버튼을 클릭합니다.
6. 다시 앱을 실행하면 정상적으로 열립니다.

### 방법 3: 터미널 명령어 (위 방법이 안 될 경우)
터미널을 열고 아래 명령어를 입력합니다:
```
xattr -cr /Applications/하루하루.app
```
그 후 앱을 다시 실행합니다.

## 삭제 방법

응용 프로그램 폴더에서 `하루하루.app`을 휴지통으로 이동하면 됩니다.
