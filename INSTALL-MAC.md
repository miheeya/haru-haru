# 하루하루 - macOS 설치 안내

## 1. 다운로드

아래 링크에서 **haru-haru-...-mac.zip** 파일을 다운로드합니다.

https://github.com/miheeya/haru-haru/releases/latest

## 2. 설치

1. 다운로드한 zip 파일을 더블클릭하여 압축을 풉니다.
2. 압축 해제된 폴더 안에서 `하루하루.app`을 **응용 프로그램(Applications)** 폴더로 드래그합니다.

## 3. 보안 경고 해제 (처음 한 번만)

macOS는 App Store 외부에서 다운로드한 앱을 차단합니다.
아래 중 하나의 방법으로 해제하세요.

### 방법 A: 터미널 명령어 (가장 확실)

터미널을 열고 아래 명령어를 붙여넣기합니다:

```
xattr -cr /Applications/하루하루.app
```

그 후 앱을 더블클릭하여 실행합니다.

### 방법 B: 오른쪽 클릭으로 열기

1. `하루하루.app`을 **오른쪽 클릭** (또는 Control + 클릭)합니다.
2. 메뉴에서 **열기**를 선택합니다.
3. 경고 창에서 **열기** 버튼을 클릭합니다.

### 방법 C: 시스템 설정에서 허용

1. `하루하루.app`을 더블클릭합니다 (차단 경고가 뜹니다).
2. **시스템 설정 > 개인정보 보호 및 보안**으로 이동합니다.
3. 화면 하단의 "하루하루" 차단 메시지 옆 **확인 없이 열기**를 클릭합니다.

## 4. 접근성 권한 허용

하루하루는 현재 사용 중인 앱과 창 제목을 감지하기 위해 **접근성 권한**이 필요합니다.

1. 앱을 처음 실행하면 **"하루하루가 이 컴퓨터의 제어를 허용하도록 요청합니다"** 알림이 표시됩니다.
2. **시스템 설정 열기**를 클릭합니다.
3. **시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용**에서 `하루하루`를 켭니다.
4. 앱을 종료 후 다시 실행합니다.

> 이 권한이 없으면 활동 추적이 동작하지 않습니다.

## 삭제 방법

응용 프로그램 폴더에서 `하루하루.app`을 휴지통으로 이동하면 됩니다.
